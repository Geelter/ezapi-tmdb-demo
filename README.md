# ezapi-tmdb-demo

![coding_style](https://img.shields.io/badge/code%20style-black-000000.svg)

A streamlit app to demo ezapi-tmdb

## Environment

- Python 3.9

## Getting Started

    python -m venv .venv
    .venv\Scripts\activate
    pip install -r requirements-dev.txt
    streamlit run app.py
